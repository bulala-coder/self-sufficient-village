# Fortress OS｜自足堡壘

家庭韌性與自給自足作戰系統。

目前版本：**v6.0 Release Candidate**

Fortress OS 是以家庭為核心的本機韌性管理工具，用來盤點六大核心系統、找出跨系統缺口、安排任務與演練，並在補給或基礎設施中斷前建立可執行的準備方案。

## 核心功能

- Fortress Readiness 總準備度與跨系統警示
- 六大核心系統與壓力測試
- 任務系統與完成進度
- 技能樹與家庭能力任務
- 離線手冊
- 作戰報告
- 能力路線圖
- 家庭風險矩陣
- 情境演練
- 撤離包
- 庫存管理與保存期限追蹤

## 六大核心系統

- 水資源：飲水、生活用水、補水、淨水與停水分配
- 能源：電力、照明、通訊、必要設備與烹調燃料
- 衛生：廁所、排泄、垃圾、清潔、消毒與寵物排泄
- 醫療：急救用品、常備藥、慢性需求、聯絡人與照護方案
- 食物：家庭與寵物熱量、保存期限、烹調與配給
- 通訊：緊急聯絡、離線資訊、集合地點、紙本文件與報平安計畫

## 家庭能力

- 修繕與維護
- 做飯與配給
- 急救與照護
- 風險判斷
- 家庭分工
- 巡檢維護
- 離線知識
- 演練能力
- 心理韌性
- 安全防護與衝突避險

## 技術架構

- React
- Vite
- JavaScript / JSX
- localStorage
- GitHub Pages
- 無後端
- 無登入
- 無外部 API

Vite base 固定為 `/self-sufficient-village/`，以符合目前 GitHub Pages repo 路徑。

## 本機開發

```bash
npm install
npm run dev
```

開發伺服器預設網址：

```text
http://localhost:5173/self-sufficient-village/
```

## 檢查、建置與部署

```bash
npm run lint
npm run build
npm run deploy
```

`npm run deploy` 會將 Vite build 產生的 `dist` 發布到 GitHub Pages。請勿把本機 `dist`、`node_modules` 或系統產生檔納入一般來源提交。

## 本機資料說明

資料儲存在目前瀏覽器的 localStorage，不會同步到雲端，也不需要帳號。重新整理頁面後資料仍會保留；清除瀏覽器資料、使用不同瀏覽器或不同裝置時，既有本機紀錄不會自動出現。

## 安全聲明

Fortress OS 是家庭韌性規劃工具，不取代專業醫療、獸醫、消防、法律、電力、瓦斯、水電、建築或政府防災指引。醫療、獸醫、電力、瓦斯、高處作業、結構損壞與危險情境應尋求專業協助。

安全防護內容僅聚焦避險、防護、撤離、求救、低調與家庭安全流程，不提供武器、攻擊、陷阱或傷害他人的教學。
